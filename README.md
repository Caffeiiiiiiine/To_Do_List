# To-Do List App

This is a simple Python-based **To-Do List** application that allows users to manage their daily tasks effectively. It is designed with a clean and modular structure using **Object-Oriented Programming (OOP)** principles and supports operations like adding, removing, viewing, saving, and loading tasks.

---

## **Features**
- Add tasks with a name, description, and priority (High, Medium, Low).
- Remove tasks by name.
- View all tasks in the list with their details.
- Automatically save tasks to a CSV file (`tasks.csv`).
- Load tasks from the CSV file on startup.
- Fully modular and extensible design using Python's **OOP** principles.

---

## **Technologies Used**
- **Python 3.x**
- **CSV** module (built-in)

This project uses only Python's standard library, so no additional dependencies are required.
